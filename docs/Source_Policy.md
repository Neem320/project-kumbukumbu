
# Kenya Political Archive - Source Policy

## Purpose

The Kenya Political Archive aims to preserve Kenyan political history using verifiable, transparent, and well-documented information.

Every factual statement published by this project must be traceable to one or more sources.

---

## Source Categories

### Tier 1 - Official Sources (Highest Confidence)

Examples:

* Kenya Gazette
* Parliamentary Hansard
* IEBC official election results
* Court judgments
* Official government websites
* Acts of Parliament
* Official university records

Confidence: **Very High**

---

### Tier 2 - Reputable Publications

Examples:

* Daily Nation
* The Standard
* Business Daily
* BBC
* Reuters
* Associated Press
* Academic journals
* Published biographies

Confidence: **High**

---

### Tier 3 - Interviews

Examples:

* Television interviews
* Documentary interviews
* Podcast interviews
* Official speeches

Confidence: **Medium**

---

### Tier 4 - Community Sources

Examples:

* Blogs
* Personal websites
* Memoirs without external verification

Confidence: **Low**

---

### Tier 5 - Public Claims

Examples:

* Social media
* Anonymous allegations
* Public rumors

These should never be presented as facts. If included because they are historically significant, they must be clearly labeled as **unverified** and accompanied by context and reliable reporting about the claim itself.

Confidence: **Very Low**

---

## Every Fact Must Include

For every fact we store:

| Field       | Example                              |
| ----------- | ------------------------------------ |
| Fact        | Born in Gatundu                      |
| Source      | Kenya National Archives              |
| Author      | John Smith                           |
| Publication | Biography of Jomo Kenyatta           |
| Date        | 1975                                 |
| URL         | (if available)                       |
| Confidence  | High                                 |
| Verified    | Yes                                  |
| Notes       | Confirmed by two independent sources |

---

## Verification Rules

Before publishing:

* Check if multiple independent sources support the fact.
* Prefer primary sources over secondary ones.
* Record any disagreements between sources rather than hiding them.
* Update records if new evidence becomes available.

---

## Editorial Principles

The archive aims to document history, not promote or attack individuals.

Profiles should distinguish clearly between:

* Verified facts
* Official allegations or investigations
* Opinions
* Unverified claims
